# SWAPI
This is a Master/Detail Android app based on SWAPI (https://swapi.co/) REST API.

## Features
    - MVVM Architecture pattern
    - ROOM to persist data
    - LiveData and Data binding to display data
    - Retrofit with GSON lib to consume SWAPI
